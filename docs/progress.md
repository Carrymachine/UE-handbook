# 진행 상황

## Summary

이 문서는 핸드북 제작 범위와 현재 커버 상태를 추적한다. 공식 문서의 모든 노드를 그대로 복제하는 대신, 학습용 챕터로 얼마나 흡수됐는지를 기준으로 본다.

## 현재 상태

* 공식 `Programming with C++` 트리 기준으로 약 64개 하위 노드를 검토했다.
* 현재 핸드북 문서는 30개 이상의 md 페이지로 재구성돼 있다.
* 개발 환경, 컨테이너, 델리게이트, 아키텍처, 리플렉션 시스템의 기본 흐름은 이미 사이드바에 연결되어 있다.

## 완료된 범위

* 개발 환경과 IDE 설정 흐름
* Live Coding과 빌드 워크플로
* 컨테이너, 델리게이트, 포인터 모델
* 모듈, 자산 참조, 설정 파일, 서브시스템
* UObject, UPROPERTY, UFUNCTION, 메타데이터, 스마트 포인터

## 다음 보강 후보

* 더 많은 `.h + .cpp` 실전 예제 추가
* Gameplay Framework 전용 챕터 확장
* 생명주기와 Tick, Spawn 흐름 별도 정리
* 블루프린트와 C++ 역할 분담 예시 보강
